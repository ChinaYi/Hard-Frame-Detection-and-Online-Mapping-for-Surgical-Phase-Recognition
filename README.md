# Hard-Frame-Detection-and-Online-Mapping-for-Surgical-Phase-Recognition
The Code for M2CAI19 Paper: Hard Frame Detection and Online Mapping for Surgical Phase Recognition.


## Requirement
Pytorch version >= 0.4
python version >= 3.0
GPU Memory >= 9000mb
